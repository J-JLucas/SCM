Hi! Thanks for trying my game.

Most systems should be able to run the game out of the box.
In the rare case you need to update some C++ libraries the Unreal Engine
installer will ask you if you would like to install the missing libraries
and take care of everything for you.
Have fun!

**Controls are Standard FPS with the exception of no cycling of weapons:

Movement - wasd
Look - Mouse
Fire - Mouse1
Reload - r
Jump - Spacebar
Toggle Flashlight - f

1 - Gauss Rifle
2 - Shotgun
3 - Sniper
4 - Rocket Launcher

Sniper rifle only actions:
    (Un)Scope - Mouse2
    Toggle Nightvision - n (While scoped)

Toggle Menu - Esc/F10
